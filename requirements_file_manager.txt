# Simple File Manager Requirements
# This application uses only Python standard library modules
# No additional packages required

# Python 3.6+ required for:
# - pathlib (Python 3.4+)
# - f-strings (Python 3.6+)
# - tkinter (included with Python)

# Standard library modules used:
# - tkinter (GUI framework)
# - os (file system operations)
# - shutil (file operations)
# - stat (file statistics)
# - pathlib (path handling)
# - datetime (date/time formatting)

# Installation instructions:
# 1. Install Python 3.6 or higher
# 2. On Windows: tkinter is included with Python
# 3. On macOS: May need to install python-tk via Homebrew
# 4. On Linux: May need to install python3-tk package

# To run:
# python file_manager.py
