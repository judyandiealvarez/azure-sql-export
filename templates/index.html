Azure SQL Database Tools

Comprehensive tools for exporting, importing, and comparing Azure SQL Database schemas and data.

Export Database

Export your Azure SQL Database schema objects and table data to SQL files for backup, migration, or analysis.

  • Tables, Views, Procedures
  • Functions, Triggers
  • Table data (SQL or Binary)
  • Schema filtering
Start Export
Import Database

Import exported database objects and data into a target Azure SQL Database with smart conflict resolution.

  • Interactive mode
  • Schema comparison
  • Smart ALTER logic
  • Dependency analysis
Start Import
Compare Database

Compare exported database files with a target database to identify differences without making changes.

  • Schema comparison
  • Data comparison
  • Detailed reports
  • Export results
Start Compare
Copy Data

Copy table data from a source database to a target database using config-defined or uploaded table lists.

  • Tables from config/file
  • Optional truncate
  • Identity insert auto
  • Batch copy
